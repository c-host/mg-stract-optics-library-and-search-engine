Custom Optics Editor

Writing Optics:
1. Use the code editor below to write your optic.
2. Optics consist of a sequence of rules.
3. Each rule defines how a particular search result should be altered.
4. Rules have a Matches block and an Action block.
5. Matches can include conditions for Title, Url, Domain, etc.
6. Actions can be Boost, Downrank, or Discard.
7. You can use Like and Dislike to influence rankings.
8. You can discard non-matching results to exclude them from the results.
9. More information on writing optics can be found here.

Using Optics:
1. Save your optic.
2. A unique URL will be generated and used as the new Custom Optic.
3. Test your optic using the "Test Optic" button below.
4. Use the "Search" button to perform a new search with your optic.
5. You can edit your optic and save it to update the Custom Optic.
